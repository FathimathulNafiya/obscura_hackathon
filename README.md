# Job Orbit

Job Orbit is a comprehensive job application system connecting freshers and recruiters.

## Features

- **Authentication:** Secure Login/Signup for Candidates and Recruiters.
- **Role-Based Portals:**
  - **Candidates (Freshers):** Search jobs, apply, view application status, and build a resume profile.
  - **Recruiters:** Post jobs, view applications, shortlist candidates, and schedule interviews.
- **Dashboard:** Interactive dashboards with real-time updates.
- **UI/UX:** Modern dark-themed, glassmorphism design.

## Tech Stack

- **Frontend:** React, Vite
- **Backend:** Node.js, Express
- **Database:** JSON-based (File System)

## Setup

1. **Backend:**
   ```sh
   cd backend
   node server.js
   ```

2. **Frontend:**
   ```sh
   cd frontend
   npm install
   npm run dev
   ```
